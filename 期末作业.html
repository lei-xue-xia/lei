CSS语法基础

CSS（层叠样式表）是用于描述网页呈现方式的语言。通过CSS，我们可以控制网页的布局、颜色、字体等视觉表现。CSS语法由选择器和声明块组成，选择器用于指定要样式化的HTML元素，声明块包含一个或多个用分号分隔的声明。

CSS的主要特点包括层叠性（多个样式规则可以应用于同一元素）、继承性（某些样式属性可以从父元素继承到子元素）和优先级（不同选择器具有不同的权重）。

选择符

CSS选择符用于选择要应用样式的HTML元素。常见的选择符类型包括：

元素选择器：通过HTML元素名称选择，如 p { } 选择所有段落
类选择器：通过类名选择，如 .abstract { }
ID选择器：通过元素的id属性选择，如 #CSSbase { }
伪类选择器：用于定义元素的特殊状态，如 a:hover { }

CSS定位：相对定位与绝对定位

定位概念解析

CSS定位是控制元素在页面上位置的重要机制，主要有以下几种定位方式：

  • 静态定位（static）：默认定位方式，元素按照正常文档流排列
  • 相对定位（relative）：相对于元素在正常文档流中的位置进行偏移
  • 绝对定位（absolute）：相对于最近的非static定位祖先元素进行定位
  • 固定定位（fixed）：相对于浏览器窗口进行定位
  • 粘性定位（sticky）：在特定阈值前是相对定位，之后变为固定定位

关键区别解析

  • 位置参考点：相对定位相对于元素自身原始位置，绝对定位相对于最近的定位祖先元素
  • 文档流影响：相对定位元素仍占据原始空间，绝对定位元素脱离文档流
  • 偏移属性：两者都使用top/right/bottom/left属性进行位置调整
  • 元素重叠：定位元素可能重叠，可使用z-index控制堆叠顺序
  • 响应式布局：相对定位更易维护响应式布局，绝对定位需谨慎处理

// 获取GitHub提交日期
async function getAllCommitDates(owner, repo) {
    const url = `https://api.github.com/repos/${owner}/${repo}/commits`;
    try {
        const response = await fetch(url);
        if (!response.ok) {
            throw new Error('网络响应失败');
        }
        const commits = await response.json();
        // 提取每个提交的日期
        const dates = commits.map(commit => commit.commit.author.date);
        return dates;
    } catch (error) {
        console.error('获取提交日期失败:', error);
        return [];
    }
}

// 显示提交日期表格
function showCommitDatesTable(dates) {
    let html = '<table><tr><th>序号</th><th>提交日期</th></tr>';
    dates.forEach((date, idx) => {
        html += `<tr><td>${idx + 1}</td><td>${date}</td></tr>`;
    });
    html += '</table>';
    document.getElementById('demo').innerHTML = html;
}

// 示例调用
getAllCommitDates('yangjh-xbmu', 'Web-develop')
    .then(dates => {
        showCommitDatesTable(dates);
    });

  • async/await：基于Promise的语法糖，使异步代码更易读写
  • fetch API：用于发起网络请求，替代传统的XMLHttpRequest
  • 错误处理：使用try/catch捕获异步操作中的错误

class Student {

    constructor(name, age, school, grade) {
        this.name = name;
        this.age = age;
        this.school = school;
        this.grade = grade;
    }
    // 方法
    learn(subject = "语文") {
        console.log(`${this.name}正在学习${subject}，真棒！`);

    }
}

xiaoming.exam();

class Undergraduate extends Student {
    constructor(name, age, school, grade, major, experience) {
        super(name, age, school, grade);
        this.major = major;
        this.experience = experience;
    }

    intern(corp) {
        console.log(`${this.name}在${corp}实习，获得经验值1000`);
        console.log(super.exam())
    }
}
