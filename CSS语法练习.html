CSS基础语法

CSS（层叠样式表）由选择器和声明块组成：

基本语法结构

选择器 {
  属性: 值;
  属性: 值;
}

示例：

这段文本将应用下面的CSS样式

实际应用

/* 选择所有段落元素 */
p {
  color: #3498db;
  font-size: 18px;
  font-weight: bold;
}

/* 选择类名为 demo-box 的元素 */
.demo-box {
  background-color: #f1c40f;
  padding: 20px;
  border-radius: 8px;
}

CSS选择器类型

基本选择器

/* 元素选择器 */
p { ... }

/* 类选择器 */
.className { ... }

/* ID选择器 */
#elementId { ... }

/* 通用选择器 */
* { ... }

组合选择器

/* 后代选择器 */
div p { ... }

/* 子元素选择器 */
div > p { ... }

/* 相邻兄弟选择器 */
div + p { ... }

/* 通用兄弟选择器 */
div ~ p { ... }

CSS盒模型

每个HTML元素都可以看作是一个盒子，包含内容(content)、内边距(padding)、边框(border)和外边距(margin)。

margin border padding content
内容区域

盒模型代码示例

.box {
  /* 内容尺寸 */
  width: 200px;
  height: 150px;
  
  /* 内边距 */
  padding: 20px;
  
  /* 边框 */
  border: 10px solid #3498db;
  
  /* 外边距 */
  margin: 20px;
  
  background-color: #2ecc71;
}

定位与布局

定位类型

/* 静态定位（默认） */
position: static;

/* 相对定位 */
position: relative;
top: 10px;
left: 20px;

/* 绝对定位 */
position: absolute;
top: 0;
right: 0;

/* 固定定位 */
position: fixed;
bottom: 20px;
right: 20px;

/* 粘性定位 */
position: sticky;
top: 0;

伪类与伪元素

伪类示例

正常状态 悬停状态 点击状态

常用伪类

/* 链接状态 */
a:link { ... }    /* 未访问的链接 */
a:visited { ... } /* 已访问的链接 */
a:hover { ... }   /* 鼠标悬停 */
a:active { ... }  /* 点击时 */

/* 表单元素 */
input:focus { ... }  /* 获得焦点时 */
input:disabled { ... } /* 禁用状态 */
input:checked { ... } /* 选中状态 */

/* 结构伪类 */
li:first-child { ... }  /* 第一个子元素 */
li:last-child { ... }   /* 最后一个子元素 */
li:nth-child(2) { ... } /* 第2个子元素 */

过渡与动画

将鼠标悬停在方块上查看效果

过渡效果

.box {
  transition: all 0.3s ease;
}

.box:hover {
  transform: scale(1.2) rotate(45deg);
  background-color: #3498db;
}

关键帧动画

/* 定义动画 */
@keyframes pulse {
  0% {
    transform: scale(1);
    opacity: 1;
  }
  50% {
    transform: scale(1.2);
    opacity: 0.7;
  }
  100% {
    transform: scale(1);
    opacity: 1;
  }
}

/* 应用动画 */
.element {
  animation: pulse 2s infinite;
}
